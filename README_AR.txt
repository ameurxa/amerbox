AmerBox - حزمة جاهزة للنشر على Vercel (نسخة عربية)

هذه حزمة سريعة التحضير تحتوي على صفحات واجهة الزبون ولوحة الإدارة
مجهزة بالتصميم (أزرق + رمادي)، شعار نصي، وصل PDF (قالب)، وملفات إعداد Vercel.

محتويات الحزمة:
- amerbox_client/: واجهة الزبون (index.html, track.html, assets)
- amerbox_admin/: لوحة الإدارة (login.html, index.html, assets)
- vercel.json : إعدادات النشر على Vercel كخدمة صفحات ثابتة
- .env.example : مثال متغيرات البيئة (ضع DATABASE_URL الخاص بـ Supabase هنا)
- README_AR.txt : هذا الملف

ملاحظة مهمة:
- هذه حزمة "سريعة" (static skeleton) لتسهيل رفعها إلى Vercel كاختبار.
- لنسخة Django كاملة مع إدارة قواعد البيانات وملفات Python يجب استبدال هذه الملفات بمشروع Django الكامل
  أو رفع المشروع Django إلى Vercel باستخدام GitHub + إعدادات الـ Build (تتطلب خطوات إضافية).
- عند ربط Supabase: ضع قيمة DATABASE_URL في متغيرات البيئة داخل Vercel أو داخل ملف .env وغيّر الإعدادات في مشروعك.
